# 🎮 PokéFinder

A web app for looking up information on any Pokémon using the PokéAPI. 
Enter a Pokémon’s name and view its card, including its stats, types and even its official sound.

## 🚀 Features

- Dynamic search by name using the Enter key or a button.
- Card generated dynamically via DOM manipulation.
- Background colour adapted to the Pokémon’s type.
- Support for dual-type Pokémon.
- 3D tilt effect on mouseover, implemented using pure JavaScript.
- Official Pokémon sound playable from the card.
- Error handling with a message visible to the user.
- Card entry animation using CSS.

## 📚 What I learned

- Using APIs with `fetch` and `async/await`.
- Advanced DOM manipulation to generate dynamic HTML.
- Mouse events to create 3D effects using `getBoundingClientRect`.
- The browser’s Audio API for playing sounds.
- Handling nested data in objects and arrays.

## 🔮 Future Improvements

- Dark mode.
- Rebuild it using React as a migration exercise.
- Add more stats and detailed information about Pokémon.
- Search by Pokédex number as well as by name.
- Improved support for Pokémon with alternate forms.
